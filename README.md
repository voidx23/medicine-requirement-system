# Medicine Requirement List Management System

## MVP Features

### Includes
- **Add Supplier**: Ability to add new suppliers.
- **Add Medicine**: Ability to add new medicines.
- **Daily Requirement List**: Manage daily lists of required medicines.
- **Prevent Duplicate**: Validation to prevent duplicate medicine entries per day.
- **Delete Entry**: Ability to remove medicine entries from the list.
- **Generate PDF**: Generate supplier-wise PDF reports.

### Excludes (for now)
- Authentication
- Quantity tracking
- Approval workflow
